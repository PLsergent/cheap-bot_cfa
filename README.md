# cheap-bot_cfa
React native project, cheap bot for social media.

## Generate docs
```
cd cheap_bot
npm i -g react-docgen
react-docgen ./code/components/* | ./docs/buildDocs.sh
```

Docs available here :

https://plsergent.github.io/cheap-bot_cfa/

## Run project
```
cd cheap_bot/code
expo start
```
